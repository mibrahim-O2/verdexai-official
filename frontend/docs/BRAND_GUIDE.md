\# VerdexAI Brand Guide



\## Color Palette

\- \*\*Primary (Brand Blue)\*\*: oklch(0.52 0.22 258) — light mode / oklch(0.68 0.2 258) — dark mode

\- \*\*Accent (Electric Blue)\*\*: oklch(0.55 0.18 230) — light mode / oklch(0.62 0.16 230) — dark mode

\- \*\*Background\*\*: White (light) / Deep navy oklch(0.16 0.04 250) (dark)

\- \*\*Destructive\*\*: Standard red, unchanged from base theme



\## Typography

\- \*\*Sans\*\*: Geist (with fallback)

\- \*\*Mono\*\*: Geist Mono (with fallback) — used for code/test sections



\## Logo Concept

\- Wordmark "VerdexAI" — geometric sans, primary blue, optional accent-blue gradient on "AI"

\- Icon concept: abstract interlocking "V" formed by two angular strokes, suggesting verification/growth (Verdex = "verify" + "index/apex")

\- No literal hiring/recruitment iconography (avoid dating the brand to one vertical)



\## Usage Rules

\- Dark mode is the primary/default brand experience (matches founder's known aesthetic preference for navy/electric-blue dark UI)

\- Light mode must remain fully accessible — do not drop contrast ratios below WCAG AA

\- Primary blue reserved for CTAs, active states, links

\- Accent blue reserved for secondary highlights, charts, hover states
