# VerdexAI Brand Guide

## Color Palette
- **Primary (Brand Blue)**: oklch(0.52 0.22 258) light / oklch(0.68 0.2 258) dark
- **Accent (Electric Blue)**: oklch(0.55 0.18 230) light / oklch(0.62 0.16 230) dark
- **Background**: White (light) / Deep navy oklch(0.16 0.04 250) (dark)
- **Destructive**: Standard red, unchanged from base theme

## Typography
- **Sans**: Geist (with fallback)
- **Mono**: Geist Mono (with fallback) — used for code/test sections

## Logo Concept
- Wordmark "VerdexAI" — geometric sans, primary blue, accent blue on "AI"
- Icon: abstract angular "V" (verify + apex), no literal hiring iconography

## Usage Rules
- Dark mode is the primary brand experience
- Light mode must remain WCAG AA accessible
- Primary blue: CTAs, active states, links
- Accent blue: secondary highlights, charts, hover states
